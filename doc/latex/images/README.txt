Thư mục này dùng để lưu trữ các hình ảnh cho tài liệu LaTeX.

Các hình ảnh cần chuẩn bị:
1. ctu_logo.png - Logo Trường Đại học Cần Thơ (sử dụng ở trang bìa).
2. component-diagram.png - Sơ đồ thành phần (được thiết kế trong diagrams/puml/component-diagram.puml).
3. class-diagram.png - Sơ đồ lớp (được thiết kế trong diagrams/puml/class-diagram.puml).
4. use-case-diagram.png - Sơ đồ Use Case tổng quan (được thiết kế trong diagrams/puml/use-case-diagram.puml).
5. sequence-diagrams.png - Các sơ đồ tuần tự (được thiết kế trong diagrams/puml/sequence-diagrams.puml).
6. activity-diagrams.png - Các sơ đồ hoạt động (được thiết kế trong diagrams/puml/activity-diagrams.puml).

Cách chuyển đổi:
- Với các file .puml (PlantUML), bạn có thể dùng extension VS Code hoặc command line PlantUML để export sang PNG.
- Với các file Mermaid trong thư mục diagrams/mermaid, bạn có thể export trực tiếp trên Mermaid Live Editor hoặc Mermaid CLI.
- Sau khi có ảnh, hãy lưu vào thư mục này với đúng tên file tương ứng để tài liệu LaTeX tự động tải ảnh vào báo cáo.
