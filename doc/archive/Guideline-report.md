# Regulations on the Format and Content of the Report

The report must follow the regulations below.

## 1. Formatting the Report

- **Chapter title (Heading 1):** Font Arial, Size 14, Bold, Uppercase.
- **Section title (Heading 2):** Font Arial, Size 13, Bold, Uppercase.
- **Subsection title (Heading 3):** Font Arial, Size 13, Bold & Italic, Lowercase.
- **Sub-subsection title (Heading 4):** Font Arial, Size 13, Italic, Lowercase.
- **Normal text:** Times New Roman, Size 13, lowercase.
- Use the "Format/Style" feature in your editor to define these styles.
- The table of contents should include four levels (Heading 1–4) and be generated automatically.
- Number chapters and sections in outline style (e.g., `1.2.1`).
- Ensure content is clear, concise, and free of spelling errors.

## 2. Layout and Content of the Report

### 2.1 Cover Page
Present the cover page formally and include:

- Name of the university (with logo).
- Class name (major, cohort, and course class).
- Type: Project – Basic Topics.
- Title of the project.
- Academic degree and full name of the advisor.
- Student ID and full names of the students involved.
- Semester and academic year.

### 2.2 Abstract

Write a concise summary (150–200 words) that includes:

- The primary objective or goal of the project.
- A brief description of the methodology or approach.
- Key results or findings.
- The significance or impact of the project.

### 2.3 Table of Contents

- Use an automated Table of Contents that lists headings to four levels with page numbers.

### 2.4 List of Tables and List of Figures

- Include a **List of Tables** and a **List of Figures**. Number illustrations sequentially (Figure 1, Figure 2, ...).

### 2.5 List of Abbreviations

- Provide a list of abbreviations and their full forms for reader clarity and consistency.
