REGULATIONS ON THE FORMAT AND CONTENT 
OF THE  REPORT

The report must follow the general regulations outlined below.
FORMATTING THE REPORT
Chapter title (Heading 1): Font Arial, Size 14, Bold, Uppercase.
Section title (Heading 2): Font Arial, Size 13, Bold, Uppercase.
Subsection title (Heading 3): Font Arial, Size 13, Bold, Italic, Lowercase.
Sub-subsection title (Heading 4): Font Arial, Size 13, Italic, Lowercase.
Normal text: Font Times New Roman, Size 13, Lowercase. 
Use the “Format/Style” function to define these styles. 
The table of contents should be presented in four levels (Heading 1, 2, 3, 4). Use the "Table of Contents" tool to generate an automatic table of contents.
Number chapters and sections in an Outline style (e.g., 1.2.1 refers to sub-section 1 in section 2 of chapter 1). 
The content should be clear, concise, and free from spelling errors.
THE LAYOUT AND CONTENT OF THE REPORT
COVER PAGE
Present formally with all the following information:
-	Name of the university (with logo).
-	Class name (major, cohort, and course class).
-	Type: Project – Basic Topics.
-	Title of the project.
-	Academic degree and full name of the advisor.
-	Student ID, full name of the students involved.
-	Semester and academic year of the report.
ABSTRACT
Write a concise summary of the entire project. The abstract should include:
-	The primary objective or goal of the project.
-	A brief description of the methodology or approach taken.
-	Key results or findings from the project.
-	The significance or impact of the project.
-	Length: 150-200 words, written last but presented at the beginning.
TABLE OF CONTENTS
Use an automated table of contents feature to ensure consistency. It should list all main sections and subsections with page numbers. The table of contents should be presented in four levels (Heading 1, 2, 3, 4). 
LIST OF TABLES
A list of all the tables (e.g., diagrams, charts, graphs, or illustrations) included in the report. This helps readers quickly locate specific figures within the document and provides an overview of the visual content included. Each figure is numbered sequentially (e.g., Figure 1, Figure 2, etc.) to maintain order.
LIST OF FIGURES
A list of all the figures (e.g., diagrams, charts, graphs, or illustrations) included in the report. This helps readers quickly locate specific figures within the document and provides an overview of the visual content included. Each figure is numbered sequentially (e.g., Figure 1, Figure 2, etc.) to maintain order.
LIST OF ABBREVIATIONS
A list of all the abbreviations and acronyms used in the report, along with their full forms. This table is used to: ensure that readers understand the shortened terms without confusion, especially if they are unfamiliar with the terminology; maintain consistency in the use of abbreviations, promoting uniformity in the report.
